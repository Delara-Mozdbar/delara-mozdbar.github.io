Experiment 01

Human-AI Co-Creation

Research focus: Exploring how to maintain artistic authenticity in the transition from analog media to generative tools.

My hand-drawn sketches serve as base inputs for generative AI systems. The question isn't whether AI can enhance them — but whether the output still carries the original human intent. Three experiments below pair each analog source with its AI-augmented result.

Hypothesis

If the original composition, line weight, and color decisions are preserved as constraints, AI enhancement will amplify the artistic intent rather than replace it.

Method

Each hand drawing was photographed and used as a base input across multiple generative models. The same source image was tested with varied prompt structures — from minimal ("enhance this sketch") to constrained ("preserve line weight, composition, and color palette").

Observations

Minimal prompts produced technically impressive but generic results — the AI's style overwrote mine. Constrained prompts preserved authorship: the output felt like an evolution of my drawing, not a replacement. The tighter the constraints, the more the result honored the original intent.

What's next

Testing the same methodology with animation tools — can motion be added to analog drawings while preserving their hand-made character?

Key finding

Preserving line weight and composition from the analog source maintains authorship — the AI amplifies rather than replaces the artistic decision.

Experiment 02

Vibe-Coding: From Design Intent to Working Product

Research focus: Investigating whether AI coding tools can translate a designer's intent directly into functional interfaces — accelerating the journey from concept to working prototype.

A planning and time-management web project built with HTML, CSS, and AI-assisted development. Instead of handing off designs to engineers, I collaborated directly with AI coding tools — testing how far design intent survives the translation to functional code.

Hypothesis

A designer with engineering fluency can direct AI coding tools to produce production-quality interfaces — without writing every line manually.

Method

Designed the planning interface concept first, then collaborated with AI coding tools (Claude, ChatGPT) through iterative prompts. Each iteration was reviewed against design intent: spacing, hierarchy, interaction states, and responsive behavior.

Observations

AI accelerated the build dramatically — but only because I could evaluate the code quality and correct the direction. Vague prompts produced generic layouts; specific design-language prompts ("8px spacing system, muted borders, single accent color") produced results faithful to intent. The bottleneck isn't the AI — it's the clarity of the design direction.

What's next

Expanding the project with data persistence and testing whether AI can maintain design-system consistency across a growing codebase.

Key finding

AI coding tools excel at translating clear design intent into working code — but the critical decisions remain fundamentally human. The designer who can direct AI fluently ships faster without sacrificing quality.
